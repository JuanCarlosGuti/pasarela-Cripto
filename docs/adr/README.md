# Registros de Decisiones de Arquitectura (ADRs)

Un ADR (*Architecture Decision Record*) documenta una decisión importante de
arquitectura: el contexto, la decisión tomada y sus consecuencias. Sirve para que
cualquiera (incluido tu yo del futuro) entienda **por qué** el sistema es como es.

## Formato

Cada ADR sigue esta estructura:
- **Estado:** propuesto / aceptado / reemplazado.
- **Contexto:** qué problema o situación motivó la decisión.
- **Decisión:** qué se decidió.
- **Consecuencias:** qué implica (a favor y en contra).
